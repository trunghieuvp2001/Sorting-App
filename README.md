# Sorting App

Sorting App là một ứng dụng được viết bằng JavaScript cho phép tạo một mảng chứa các phần tử ngẫu nhiên và sắp xếp mảng đó bằng các thuật toán sắp xếp khác nhau.

## Cài đặt

Ứng dụng này chỉ chạy trên trình duyệt web.

## Môi trường yêu cầu

- Trình duyệt web (Chrome, Firefox, Safari, etc.)

## Hướng dẫn sử dụng

1. Clone hoặc tải xuống mã nguồn của ứng dụng từ GitHub.

2. Mở file `index.html` bằng trình duyệt web.

3. Giao diện ứng dụng sẽ hiển thị. Các chức năng chính:
   - **Generate Array**: Nhấn vào nút này để tạo một mảng chứa các phần tử ngẫu nhiên.
   - **Bubble Sort**: Sắp xếp mảng bằng thuật toán Bubble Sort.
   - **Selection Sort**: Sắp xếp mảng bằng thuật toán Selection Sort.
   - **Insertion Sort**: Sắp xếp mảng bằng thuật toán Insertion Sort.
   - **Merge Sort**: Sắp xếp mảng bằng thuật toán Merge Sort.
   - **Quick Sort**: Sắp xếp mảng bằng thuật toán Quick Sort.

4. Nhấn vào nút **Generate Array** để tạo một mảng chứa 1000 phần tử ngẫu nhiên.

5. Sau đó, có thể nhấn vào các nút chức năng khác để sắp xếp mảng bằng các thuật toán khác nhau. Kết quả sẽ được hiển thị trên giao diện người dùng.

6. Nút **Generate Array** để tạo một mảng mới và tiếp tục sắp xếp.
